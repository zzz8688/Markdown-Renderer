自动补齐功能说明
只能在最后一行补齐
 关掉「自动修复Markdown」：只会对未闭合 ``` 做保护，其它统统不改。
 打开时：所有修复规则生效。

为什么开关关了还能自动补齐？
关掉开关时，我们确实“不会动公式和 table”
1.remark-math 的行为是：遇到 $$，会把后面的内容一直吃到下一个 $$ 或文件结尾，作为一个公式块。
2.GFM 表格由 remark-gfm 解析，它的规则也比较宽松：末尾少一个 |，大多数情况下依然能被当成一行表格来处理；
3.remark-parse 对 ``` 的行为：
遇到一个起始 ，如果一直到文件末尾都没再遇到闭合 ：
它会把后面的所有内容都当成这个代码块的一部分。
这不是“自动补上一个 ```”，而是“把剩下的全吞进去”。
